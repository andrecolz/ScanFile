# ScanFile CMD

```
static void Main(string[] args)
```
- args[0] ---> path di ricerca
- args[1] ---> tipo file export
- args[2] ---> nome file export
- args[3] ---> percorso file export
- args[4] ---> percorso 1° file di confronto
- args[5] ---> percorso 2° file di confronto
